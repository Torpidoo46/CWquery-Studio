Cloudwatch Query Studio

Query Command Guide
fields

Retrieve one or more log fields. Supports functions like abs(), sqrt(), strlen(), etc.

fields @log, @logStream, @message, @timestamp
filter

Retrieve logs based on conditions with operators or regex.

filter @message like /(?i)(Exception|error|fail|5dd)/
filterIndex

Limit query to indexed data only.

filterIndex field_index = value
stats

Calculate aggregate statistics like count(), sum(), avg().

stats count() by bin(5m)
sort

Sort results by a field.

sort @timestamp asc
limit

Limit number of log events returned.

limit 10
parse

Create fields from message patterns.

parse '* - * [*] "* * *" * *' as host, identity, ...
dedup

Remove duplicate log entries.

dedup @logstream, @timestamp
pattern

Identify and collapse similar logs.

pattern @message
diff

Compare log pattern changes across time.

diff
unnest

Flatten a list or JSON array.

unnest field into unnested_field
