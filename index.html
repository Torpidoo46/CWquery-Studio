Cloudwatch Query Studio

Command Guide

fields

Retrieve one or more log fields using functions and operations.

fields @log, @logStream, @message, @timestamp

filter

Retrieve log fields based on one or more conditions.

filter @message like /(?i)(Exception|error|fail|5dd)/

filterIndex

Limit query to indexed data only using specified field index.

filterIndex field_index = value

stats

Calculate aggregate statistics such as count(), avg(), etc.

stats count() by bin(5m)

sort

Sort the log fields in ascending or descending order.

sort @timestamp asc

limit

Limit the number of log events returned by a query.

limit 10

parse

Create ephemeral fields for further processing in query.

parse '* - * [*] "* * *" * *' as host, identity, etc.

dedup

Removes duplicate results based on provided fields.

dedup @logstream, @timestamp

pattern

Identify and collapse logs into structural patterns.

filter @message like /ERROR/ | pattern @message

diff

Compare changes in patterns between time ranges.

fields @message | pattern @message | diff

unnest

Flatten a list into multiple records.

unnest field into unnested_field
